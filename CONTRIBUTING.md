# Contributing

Thanks for helping make AI-agent-assisted repositories safer and easier to maintain.

## Local Setup

```bash
npm install
npm run check
```

## Pull Requests

- Keep changes small and focused.
- Add or update a fixture when changing scanner behavior.
- Include tests for new rules.
- Never include real secrets, private logs, private URLs, cookies, or browser profiles.
- Explain user-facing output changes in the PR summary.

## Good First Issues

Good first contributions usually fit one of these shapes:

- Add a fixture for a real repository pattern.
- Improve report wording.
- Add a scanner for one narrow, documented risk.
- Improve README or docs clarity.
